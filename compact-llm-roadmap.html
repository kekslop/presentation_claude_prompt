Фаза 1: MVP 3-4 мес
  • Реализация чат-поддержки 1 линии (90% автоматизация)
  • Минимальная версия платформы роутинга на базе LiteLLM
  • Интеграция с 1-2 критичными системами через API
  • Бюджет: 5-10 млн руб
Фаза 2: Фундамент платформы 6-8 мес
  • Разработка полноценной платформы роутинга запросов
  • Интеграция с ключевыми системами (CRM, АБС)
  • Система управления промптами версия 1.0
  • Подключение 3-5 процессов с высокой автономностью
  • Бюджет: 30-50 млн руб
Фаза 3: Масштабирование 12-18 мес
  • Полная интеграция со всеми критичными системами
  • Расширенная инфраструктура мониторинга
  • Механизмы постоянного улучшения моделей
  • Охват всех процессов высокой и средней автономности
  • Бюджет: 80-120 млн руб
Фаза 4: Системная трансформация 18-36 мес
  • Разработка AI-нативных банковских продуктов
  • Глубокая интеграция AI в процессы принятия решений
  • Создание новых каналов взаимодействия на базе AI
  • Персонализация на уровне отдельного клиента
  • Бюджет: 150-200 млн руб
Модель финансирования: Бюджет формируется через комбинацию: 60-70% из централизованного инвестиционного бюджета и 30-40% из операционных бюджетов подразделений-бенефициаров. Применяется "плавающий бюджет" с автоматическим выделением средств при достижении KPI.
